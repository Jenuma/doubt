It looks like you're using Internet Explorer.

Some features of Become Rampant may not work correctly if you continue.

Why not?

Internet Explorer needs special, old code to be compatible with modern web technology - the same technology we utilize to bring you a great experience. The reason we don't go the extra mile to be compatible with Internet Explorer is because we don't want you to be at risk. Many versions of this browser have security vulnerabilities that hackers can exploit to steal information, money, or even your identity with.

What can I do to be safer online?

We recommend you download a different web browser, such as:

  • Google Chrome
  • Mozilla Firefox
  • Opera

All of these options are free to download and use indefinitely and are safer and easier to use than Internet Explorer. When you return to becomerampant.com, you will no longer see this message.

What is a web browser?

A web browser is the program (or application, or "app") that you use to go to websites. If you bought a PC with Windows, it came preloaded with Internet Explorer. However, it is highly recommended to download a different browser from the list above to stay safe andexperience all the modern technology the Internet has to offer. For more information, click here.

What if I can't install a new browser on this computer?

We understand that there may be a situation in which you can't download a new web browser - for example, if you are using a work or library computer that prevents users from installing new programs. You can still proceed to Become Rampant if that is the case, but beware that many features may not work, and the site may be unsightly to view.
